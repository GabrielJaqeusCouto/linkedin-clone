## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone https://github.com/React-Bootcamp-WoMarkersCode/certificate-generator
```

Acesse a pasta clonada:

```
cd 
```

```
npm install
```

```
npm start
```
